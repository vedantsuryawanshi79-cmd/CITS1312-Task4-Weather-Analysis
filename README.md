## 📌 Project Scope

This project analyzes weather patterns for Hyderabad across 12 months. It covers temperature trends, rainfall and humidity data, seasonal comparisons, and weather event distribution to understand climate behavior throughout the year.

---

## 📊 Features

- Monthly Temperature Trends — Max, Min, Avg (Line Chart)
- Rainfall and Humidity Combined Chart (Bar + Line)
- Seasonal Average Temperature Comparison (Bar Chart)
- Weather Event Distribution (Doughnut Chart)
- Monthly Statistics Table

---

## 🔍 Key Insights

- **Peak Temperature:** 42°C in May 2026
- **Minimum Temperature:** 11°C in January 2026
- **Annual Rainfall:** 812mm (↑ 6% vs last year)
- **Average Humidity:** 64% (Peak 91% in August)
